    مستقیم

    روش های اتصال مستقیم به سرور, در این روش ها تنها به یک سرور خارجی نیاز دارید. روش های اصلی اتصال مستقیم به سرور عبارتند از:

    1. V2Ray با کانفیگ های مختلف:
    2. VLESS + TLS + ECH
    3. VLESS با فرگمنت
    4. VLESS + QUIC

    5. VLESS با IPv6

    6. Brook: پروتکل ساده برای دور زدن فیلترینگ

    7. Amnezia VPN: مناسب برای ترید با پینگ پایین

    8. Hysteria 2 و TUIC: پروتکل های با سرعت بالا و پینگ پایین

    9. Shadowsocks با Cloak server

    10. ترکیب V2Ray با Cloudflare:

    11. استفاده از ساب دامنه ها و CNAME

    12. استفاده از IPv6 و Cloudflare

    13. ترکیب VLESS با Wireguard

    14. استفاده از Cloudflare Tunnel بدون نیاز به دامنه

    15. هیدیفای (Hiddify): پنل مدیریت با قابلیت ساخت انواع کانفیگ

    16. ترکیب Gcore با Cloudflare

    17. استفاده از Fastly CDN

    18. HTTP Upgrade: روش جدید برای رفع اختلال اینترنت

    19. ترکیب Nginx با HTTP Upgrade

    این روش ها عموماً برای دور زدن فیلترینگ، افزایش سرعت و کاهش پینگ طراحی شده اند. برخی نیاز به دامنه دارند و برخی بدون دامنه قابل استفاده هستند. انتخاب روش مناسب به نیازهای خاص کاربر و شرایط اینترنت در منطقه بستگی دارد.
