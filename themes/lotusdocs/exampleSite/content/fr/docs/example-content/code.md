## Code Block Examples

Code block language examples

### Go

```go
package main

import "fmt"

func main() {
    fmt.Println("hello world")
}
```

### HTML

```html
<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>

<p>My first paragraph.</p>

</body>
</html>
```
